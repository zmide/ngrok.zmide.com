简单使用
ngrok 服务可以分配给你一个域名让你本地的 web 项目提供给外网访问，特别适合向别人展示你本机的 web demo 以及调试一些远程的 API ( 比如微信公众号，企业号的开发 )

Windows

  • 下载 Windows 版本的客户端并解压
  • 在命令行下进入到 ngrok 客户端目录下
  • 执行 ngrok -config=ngrok.cfg -subdomain test 80  ( test 是你连接注册的的域名前缀，如果连接失败可以换个前缀 )
  • 如果开启成功，就可以使用（ http://test.durl.ga/ ）域名来访问你本机的 127.0.0.1:80 的服务

    • Linux

  • 下载 linux 版本的客户端并解压
    • （ 注意：要给 ngrok 文件的可执行权限 ） sudo chmod +x nginx
  • 在命令行下进入到 ngrok 目录下
  • 执行 ./ngrok -config=ngrok.cfg -subdomain test 80  ( test 是你连接注册的的域名前缀，如果连接失败可以换个前缀 )
  • 如果开启成功，就可以使用（ http://test.durl.ga/ ）域名来访问你本机的 127.0.0.1:80 的服务

    • Mac os

  • 下载 Mac 版本的客户端并解压
    • （ 注意：要给 ngrok 文件的可执行权限 ） sudo chmod +x nginx
  • 在命令行下进入到 ngrok 目录下
  • 执行 ./ngrok -config=ngrok.cfg -subdomain test 80  ( test 是你连接注册的的域名前缀，如果连接失败可以换个前缀 )
  • 如果开启成功，就可以使用（ http://test.durl.ga/ ）域名来访问你本机的 127.0.0.1:80 的服务

    • 使用自己的域名？

  • 将自己的域名 A 解析到 IP 152.32.128.230
  • 在命令行下进入到 ngrok 目录下
  • 执行 ./ngrok -config=ngrok.cfg -hostname xxx.xxx.xxx 80  ( xxx.xxx.xxx 是你自己的顶级域名 )
